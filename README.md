## What's here

Sublingual OS — an operating system for your thinking.

```
sos-base     the body      Arch + Hyprland. Installs on hardware.
sos-shell    the nerves    Agent daemon. Operates the body. (coming)
sos-kernel   the seed      Protocol that makes it think. (coming)
sos-os       the mind      Runtime and agentic harness. (coming)
```

**sos-base** is the foundation layer — what you're looking at. The install, the configs, the packages.

## Install

On a fresh Arch Linux system (Limine + Btrfs via [archinstall](https://wiki.archlinux.org/title/Archinstall)):

```bash
git clone https://github.com/wearesublingual/sos-base.git ~/.local/share/sublingual-os
cd ~/.local/share/sublingual-os
SUBLINGUAL_OS_ONLINE_INSTALL=true bash install.sh
```

Reboot...and you're in the future.

## Where this is going

**sos-base** — the body. Linux that installs and runs. *(you are here)*

**sos-shell** — the nerves. An always-on agent that operates the machine. System health, troubleshooting, environment setup — the reason Linux stops being a hobby and starts being infrastructure. Your machine has a partner who knows it deeply enough that you don't have to.

**sos-kernel** — the seed. The protocol that makes it all think. Plant it on any hardware, and a thinking system grows. Not configuration — DNA.

**sos-os** — the mind. The runtime and agentic harness that composes sos-base (body), sos-shell (nerves), and sos-kernel (seed) into a generative computing system.
